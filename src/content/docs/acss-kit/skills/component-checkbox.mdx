## CSS variable prefix

Checkbox tokens follow the `--chk-*` naming convention, e.g. `--chk-bg`, `--chk-border`, `--chk-indicator`.

## Accessibility

Checkbox uses `aria-checked` (including `"mixed"` for indeterminate state) and `aria-disabled="true"` for the disabled state. Labels are always associated via `htmlFor` or wrapping.
