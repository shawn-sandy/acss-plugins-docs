Consumer guides for the 15 components generated by `acss-kit`. Each guide covers how to add the component, how to import it, its props, copy-paste examples, theming CSS variables, and accessibility notes.

These are **consumer** guides — documentation written for developers using the generated components in their own projects. The canonical skill references (generation contracts, SCSS templates) live in `acss-kit/skills/component-*`.

## First time here?

1. Run `/setup` once — installs `sass`, creates the `ui.tsx` foundation, writes `.acss-target.json`, and seeds a starter theme.
2. Add a component with `/kit-add <Name>`, or install everything at once with `/kit-sync`.
3. Import the generated `.tsx` + `.scss` pair and use it.

See [`/kit-add`](/acss-plugins-docs/acss-kit/commands/kit-add) for the full generation workflow and [`/kit-sync`](/acss-plugins-docs/acss-kit/commands/kit-sync) to install all components at once.
