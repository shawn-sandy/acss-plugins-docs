<Aside type="caution">
  **`acss-utilities` is deprecated.** All `/utility-*` commands and the
  `utilities` skill are now part of **acss-kit v1.0.0**.

```text
/plugin uninstall acss-utilities@shawn-sandy-agentic-acss-plugins
/plugin update acss-kit@shawn-sandy-agentic-acss-plugins
```

See the [Migration Guide](/acss-plugins-docs/acss-kit/migration-v1) for the
full upgrade steps.

</Aside>

This page is preserved for reference. The canonical documentation for the
`utilities` skill is now at [`acss-kit utilities skill`](/acss-plugins-docs/acss-kit/skills/utilities).

No behavior changed — the skill works identically in acss-kit v1.0.0, with the
addition that `token-bridge.css` is now regenerated automatically on every
`/theme-create` and `/theme-update` run.
