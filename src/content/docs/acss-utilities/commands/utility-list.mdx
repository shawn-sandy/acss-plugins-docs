Browse the full utility catalogue or drill into a specific family. This command is entirely read-only — it never writes files.

## Syntax

```text
/utility-list [family]
```

## Arguments

| Argument | Required | Description |
|---|---|---|
| `family` | No | Family name to inspect. If omitted, shows all families with summary info. |

## Examples

Show all families:

```text
/utility-list
```

Show all classes in a specific family:

```text
/utility-list spacing
/utility-list flex
/utility-list color-bg
/utility-list display
```

## Available family names

| Family | Description |
|---|---|
| `color-bg` | Background color utilities |
| `color-text` | Text color utilities |
| `color-border` | Border color utilities |
| `spacing` | Margin, padding, and gap |
| `display` | Display and visibility |
| `flex` | Flexbox layout |
| `grid` | CSS Grid layout |
| `type` | Typography (size, weight, style) |
| `radius` | Border radius |
| `shadow` | Box shadow |
| `position` | CSS position |
| `z-index` | Stack order |
